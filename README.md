# 解説
動画で取り扱った、ChatGPTに書かせたblenderアドオンです。

- vroid_bone_mirror_renamer.py
  - VRoidStudioからエクスポートしたVRMモデルはボーンの名前がBlenderのミラー仕様にはなっていません。それをリネームして修正するアドオンです。
- vrm_cycles_simple_materials.py
  - Blenderの標準のVRMインポータでインポートしたモデルはVRM用の複雑なシェーダーが当たっていますが、それを簡素なCycles用シェーダーに置き換えるアドオンです。
- vrm_eevee_simple_matgerials.py
  - Blenderの標準のVRMインポータでインポートしたモデルはVRM用の複雑なシェーダーが当たっていますが、それを簡素なセルルック風のEevee用シェーダーに置き換えるアドオンです。

どれもInterface languageがEnglishのBlender5.0で動作確認しています。日本語に変えると動かなくなるのはアドオンあるあるな感じなので、動かない場合はチェックお願いします。今後Blenderのバージョンがあがると動かなくなることもありそうですが、その際は必要な修正を加えるか、何かのコーディングできるAIに対応させるのがよいかと思います。
